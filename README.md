ZBX_GPRSTest
============

STB直播星GPRS模块检测工具


![效果图](http://github.com/Garfielt/ZBX_GPRSTest/raw/master/layout.png)


一个直播星GPRS定位模块的检测工具，使用51单片机实现。通过串口执行AT命令完成模块检测。主要内容包括Init、timer设定、控制及中断处理等等。串口波特率9600，具体的IO口请参照上图。
